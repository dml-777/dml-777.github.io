## Project Description
This project is a photography and digital art portfolio designed to showcase my film photography, digital photography, and interactive artwork. The website includes multiple pages, responsive layouts, dark mode functionality, and p5.js interactive sketches.

## Features
- Responsive web design
- Dark mode toggle
- Film photography gallery
- Digital photography gallery
- Interactive p5.js artwork
- Navigation across multiple pages

## Technologies Used
- HTML
- CSS
- JavaScript
- p5.js

## Challenges
One challenge during this project was organizing the file structure correctly for GitHub Pages hosting. I also worked on improving responsive layouts and implementing dark mode functionality.

## What I Learned
Through this project, I learned more about:
- GitHub Pages hosting
- Organizing file structures
- Responsive web design
- JavaScript interactions
- Debugging broken image paths
- Working with p5.js inside larger websites
