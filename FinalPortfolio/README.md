# Dona Lingga Final Portfolio

## Live Site
[View Portfolio Website](https://dml-777.github.io/FinalPortfolio/)

## Project Description
This project is a photography and digital art portfolio website created using HTML, CSS, and JavaScript.

The portfolio showcases:
- 35mm film photography
- Digital photography
- Creative coding and digital art
- An about/contact section

The goal of the project was to create a visually cohesive and professional portfolio website that reflects my artistic style and photography work.

## Features
- Responsive layout for desktop and mobile
- Dark mode/light mode toggle using JavaScript
- Organized photography galleries
- Custom styling with CSS
- Multi-page navigation

## Process
I started with my midterm portfolio concept and expanded it into a more complete and polished final project. I redesigned the layout, added more photographs, improved typography and spacing, and implemented a dark mode feature.

One of the biggest challenges was organizing the file structure and correctly linking assets and images through GitHub Pages. I also learned how to use Git and GitHub to push updates and manage my website files.

## Technologies Used
- HTML
- CSS
- JavaScript
- GitHub Pages

## Source Code Highlight
One important feature I implemented was the dark mode toggle using JavaScript. The script dynamically changes the website theme by adding and removing a dark class from the body element.
